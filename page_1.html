Типи данних, змінні й синтаксичні конструкції

Тип і значення

Значення - це конкретні данні, які присвоюються змінним. Наприклад ціле число 4, рядок "Hello world", число с плаваючою крапкою 3.14

Тип даних вказує на характеристику об'єкта (змінної) і визначає, яким чином дані можуть бути оброблені. В Python існують вбудовані типи даних такі як:

  • Int (Ціле число): Цей тип даних в Python використовується для зберігання цілих чисел. Він може містити значення, такі як 1, 2, -3 і т.д.
  • Float (Число з плаваючою комою): Float - це тип даних, призначений для зберігання дійсних чисел з плаваючою комою. Він може містити значення, такі як 3.25, -0.5 і т.д. Float використовується для представлення чисел з десятковою частиною.
  • String (Рядок): Рядок - це тип даних, призначений для зберігання послідовності символів. У Python немає окремого типу даних "char", відмінності від інших мов програмування, таких як C++ або Java. Рядок можна задати одинарними ('') або подвійними ("") лапками.
  • List (Список): Список - це тип даних, призначений для зберігання набору значень. Один список може містити різні типи значень, включаючи цілі числа, числа з плаваючою комою, рядки та інші списки. Списки використовуються для організації та зберігання даних у впорядкованій послідовності.

Змінні як об'єкт в мові Python, іменна змінних та зарезервовані слова

Змінна - це ім'я, яке використовується для зберігання та представлення даних в програмі. Змінні в програмуванні використовуються для того, щоб можна було звертатися до певних значень чи об'єктів в пам'яті комп'ютера шляхом їхніх імен.
Правила написання імені змінної:

  • Ім'я може начинитися з літер або знаків нижнього підкреслення "_"
  • Ім'я може складатися з букв, цифр або знаків нижнього "_"
  • В імені не можна використовувати спеціальні символи, наприклад /, #, або @

Інструкції

Інструкція (також відома як оператор) - це команда або дія, яку програма виконує крок за кроком. Вона вказує комп'ютеру, як виконувати певну дію або послідовність дій. В програмуванні інструкції використовуються для здійснення різних операцій, управління потоком виконання програми та взаємодії з даними.

Інструкції можуть виконувати різні дії, такі як присвоєння значень змінним, виконання математичних операцій, введення та виведення даних, управління циклами та умовними конструкціями, виклик функцій тощо.

Оператори та операнди

Оператори - це спеціальні символи, які виконують операції над значеннями та змінними та полегшують маніпулювання. Оператори можуть виконувати арифметичні, реляційні та логічні операції.

Арифметичні оператори:

  • + (додавання): Додає два числа.
  • - (віднімання): Віднімає одне число від іншого.
  • * (множення): Перемножує два числа.
  • / (ділення): Ділить одне число на інше.
  • // (цілочисельне ділення): Виконує цілочисельне ділення, ігноруючи залишок.
  • % (остача від ділення): Повертає залишок від ділення.
  • ** (піднесення до степеня): Підносить число до певного степеня.

Логічні оператори:

  • and (логічне І): Повертає True, якщо обидва умови є True.
  • or (логічне АБО): Повертає True, якщо хоча б одна з умов є True
  • not (логічне НЕ): Змінює значення на протилежне (True стає False і навпаки).

Приклади використання:

Порівняння (реляційні) оператори:

  • == (рівність): Порівнює два значення на рівність.
  • != (нерівність): Перевіряє, чи значення не рівні одне одному.
  • > (більше): Перевіряє, чи одне значення більше за інше.
  • < (менше): Перевіряє, чи одне значення менше за інше.
  • >= (більше або рівне): Перевіряє, чи одне значення більше або рівне іншому.
  • <= (менше або рівне): Перевіряє, чи одне значення менше або рівне іншому.

Приклади використання:

Присвоювання:

  • = (присвоєння значення): Використовується для присвоєння значення змінній.

Інші оператори:

  • in (перевірка на входження в колекцію): Перевіряє, чи елемент знаходиться в колекції (наприклад, списку).
  • not in (перевірка на відсутність в колекції): Перевіряє, чи елемент не знаходиться в колекції.
  • is (перевірка на тотожність об'єктів): Перевіряє, чи два об'єкти є тотожніми (вказують на один і той же об'єкт).
  • is not (перевірка на непототожність об'єктів): Перевіряє, чи два об'єкти не є тотожніми.

Приклади використання:

Операнди - це змінні або константи, з якими працюють оператори.

Пріоритети операторів

У Python оператори мають різний пріоритет, що визначає порядок їх виконання в одному виразі. Оператори з вищим пріоритетом виконуються першими. Ось загальний порядок пріоритету деяких основних операторів в Python, від найвищого до найнижчого:

  • () (Дужки) - Найвищий пріоритет. Вони використовуються для зміни порядку виразів і групування операцій.
  • ** (Піднесення до степеня) - Другий за пріоритетом.
  • *, /, %, // (Множення, ділення, остача від ділення, цілочисельне ділення) - Мають однаковий пріоритет, виконуються зліва направо.
  • +, - (Додавання, віднімання) - Мають однаковий пріоритет, виконуються зліва направо.
  • <, <=,>, >= (Оператори порівняння) - Мають менший пріоритет за побітовими операторами.
  • ==, != (Рівність, нерівність) - Мають менший пріоритет за операторами порівняння.
  • not (Логічне НЕ) - Має менший пріоритет за операторами рівності та нерівності.
  • and (Логічне І) - Має менший пріоритет за логічним НЕ.
  • or (Логічне АБО) - Найнижчий пріоритет серед логічних операторів.

Перетворення типів

У Python можливо перетворювати один тип даних в інший за допомогою функцій.

  • int(): перетворює значення на цілочисельний тип даних
  • float(): перетворює значення на тип даних із плаваючою комою
  • str(): перетворює значення на рядковий тип даних
  • bool(): перетворює значення на логічний тип даних.
    Зауважте, що такі значення буде перетворено на False: порожні послідовності та колекції: '', [], {} тощо.
    Числові нульові значення: 0, 0,0 Усе інше буде перетворено на True
  • list(): перетворює ітерований (наприклад, кортеж, рядок) на список
  • tuple(): перетворює ітерований (наприклад, список, рядок) на кортеж
  • set(): перетворює ітерований (наприклад, список, рядок) на набір
  • dict(): перетворює ітерацію пар ключ-значення (наприклад, список кортежів) у словник
