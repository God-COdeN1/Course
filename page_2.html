Поняття блоку виконання

Блок виконання - це послідовність інструкцій, які групуються разом і виконуються як єдина одиниця. В Python блоки виконання визначаються за допомогою відступів (пробіли або табуляція).

У Python відступи вказують інтерпретатору, які інструкції належать до одного блоку коду. Це важливо для структурування програм та визначення логічної ієрархії. Якщо дві інструкції мають однаковий відступ, то вони входять в один і той же блок.

Блоки виконання допомагають структурувати програми, полегшують їх розуміння та роблять код більш читабельним і підтримуваним.

Логічні вирази та оператори

Логічний вираз: Логічний вираз - це вираз, який включає умову, яка може бути істинною (True) або хибною (False). Ви можете використовувати різні оператори порівняння (наприклад, <, >, ==, !=, <=, >=) та логічні оператори (наприклад, and, or, not) для створення логічних виразів.

Обчислення логічного виразу: При обчисленні логічного виразу у Python використовується принцип "True" і "False". Якщо умова виразу є істинною, вираз обчислюється як True (або 1). Якщо умова виразу є хибною, вираз обчислюється як False (або 0).

Оператор розгалуження if ... else

Оператор if...else - це структура керування потоком програми, яка дозволяє виконувати один блок коду, якщо певна умова є істинною (True), і інший блок коду, якщо умова є хибною (False).

Основний синтаксис оператора if...else виглядає так:

Тут:

  • умова - це вираз або умова, яку ви хочете перевірити.
  • Перший блок коду (після if) виконується, якщо умова є істинною.
  • Другий блок коду (після else) виконується, якщо умова є хибною.

Приклади:

  • Перевірка числа на парність: У цьому прикладі, якщо number ділиться націло на 2 (тобто залишок від ділення дорівнює 0), то виконується перший блок коду, і виводиться "Це парне число". В іншому випадку (якщо залишок від ділення не дорівнює 0), виконується другий блок коду, і виводиться "Це непарне число".
  • Перевірка введеного користувачем числа на додатність: У цьому прикладі програма перевіряє, чи введене користувачем число є більше за 0, і виводить "Ви ввели додатне число" або "Ви ввели від'ємне або нуль" відповідно до результату перевірки.

Оператор if...else дозволяє програмі приймати рішення на основі умови, що дуже важливо для створення програм, які реагують на різні ситуації.

Вкладені конструкції

Вкладені цикли - це практика розміщення одного циклу всередині іншого циклу. Це дозволяє вам виконувати повторні дії для кожного елемента одного циклу в контексті іншого циклу. Наприклад, якщо ви маєте двовимірний масив (список списків) або таблицю, ви можете використовувати вкладені цикли для обходу кожного рядка та кожного елемента у кожному рядку.

У цьому прикладі ми маємо двовимірний список matrix, і ми використовуємо вкладений цикл for для обходу кожного рядка у matrix, а потім ще одного вкладеного циклу для обходу кожного елемента в кожному рядку. Це допомагає нам вивести всі елементи матриці.

Вкладені умовні оператори передбачають розміщення одного оператора умови всередині іншого. Це дозволяє вам створювати більш складні умовні логіки з багатьма рівнями перевірки. Наприклад, ви можете мати умовний оператор всередині іншого, щоб перевірити додаткові умови, якщо основна умова є істинною або хибною.

У цьому прикладі ми маємо вкладений оператор if...else. Спочатку ми перевіряємо, чи x більше за y. Якщо це так, то виконується перший блок коду. Потім, всередині цього блоку, ми маємо ще один оператор if, який перевіряє, чи x рівний 10. Це ще один приклад вкладеної конструкції

Вкладені структури даних можуть вкладати одну структуру даних всередину іншої. Наприклад, ви можете мати списки, що містять інші списки (списки списків) або словники, що містять інші словники (словники в словниках). Це дозволяє створювати більш складні структури даних для організації та збереження інформації.

У цьому прикладі ми маємо словники student1 та student2, які містять інформацію про студентів, а потім ми вкладаємо ці словники в список клас, створюючи таким чином структуру даних для представлення класу студентів.
