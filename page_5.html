Рядки

Кодування АЅСІІ, Unicode, Utf-8 byte-code

  • ASCII (American Standard Code for Information Interchange) - це стандарт кодування символів, який використовується для представлення тексту в комп'ютерах, комунікаційному обладнанні та інших пристроях, що використовують текст. Він був розроблений в 1960-х роках і використовує 7 біт для представлення символів, що дозволяє в цілому 128 можливих символів. Стандарт ASCII включає символи керування (наприклад, новий рядок, повернення каретки, табуляція) та друковані символи (наприклад, літери, цифри, розділові знаки).
  • Unicode - це стандарт кодування символів, який має на меті охопити всі знаки та символи, що використовуються в системах письма по всьому світу. На відміну від ASCII, який має обмежений набір символів, Unicode використовує більше бітів для представлення символів, що дозволяє набагато більшу кількість символів (понад мільйон). Кожному символу присвоюється унікальна кодова точка, що забезпечує узгоджене представлення тексту в різних мовах і шрифтах.
  • UTF-8 (Unicode Transformation Format - 8-bit) - це кодування символів змінної довжини, яке широко використовується для представлення символів Unicode в сучасних обчислювальних системах. Він використовує мінімум 8 біт (1 байт) для представлення символів ASCII (перші 128 символів, які мають ті самі кодові точки, що й ASCII), але може використовувати більше байт для символів поза діапазоном ASCII. Кількість байт, що використовується для представлення символу, залежить від його кодової точки, що дозволяє ефективно представляти найпоширеніші символи, підтримуючи при цьому повний набір символів Unicode.
  • Bytecode - це низькорівневе представлення програми, яке генерується компілятором або інтерпретатором. Він не виконується безпосередньо апаратним забезпеченням комп'ютера, а віртуальною машиною або інтерпретатором. Наприклад, Python - це інтерпретована мова програмування, і коли ви запускаєте скрипт Python, інтерпретатор Python генерує bytecode з вихідного коду. Цей bytecode потім виконується віртуальною машиною Python (PVM). Bytecode забезпечує незалежність від платформи, якщо цільова платформа має інтерпретатор або віртуальну машину, здатну виконувати bytecode.

Отже, ASCII - це стандарт кодування символів для представлення основних символів у комп'ютерах, Unicode - це всеосяжний стандарт кодування символів, що охоплює символи різних систем письма, UTF-8 - це спеціальне кодування, яке дозволяє ефективно представляти символи Unicode за допомогою кодування змінної довжини, а байткод - це проміжне представлення програми, яке використовується інтерпретаторами або віртуальними машинами для виконання.

Рядок незмінна послідовність символів

Рядок - це незмінна послідовність символів. "Незмінний" означає, що після створення рядка його вміст не може бути змінений. Будь-яка операція, яка здається, що змінює рядок, насправді створює новий рядок з потрібними змінами, залишаючи початковий рядок незмінним.

У коді вище ми створюємо рядок my_string зі значенням "Hello, World!", а потім намагаємося змінити перший символ на малу літеру 'h'. Однак це призведе до виникнення помилки типу TypeError, оскільки рядки не підтримують присвоювання елементів. Якщо потрібно змінити рядок то краще стоврити нову зміну або перезаписати стару.

В данному прикладі ми створили нову змінну і додали "h" до старої зміннохї, а позначення [1:] використовується для зрізу (slicing) рядка. Приклад [1:] означає "захопити усі елементи рядка починаючи з індексу 1 і до кінця". Більш дететально зріз рядка ми будемо розглядати пізніше.

У даному прикладі ми створили змінну `my_string` зі значенням "Hello world" і вивели її айді (ідентифікатор об'єкта) в консоль. Потім ми змінили значення змінної `my_string` на "hello world" і знову вивели її айді. Це призвело до створення нового об'єкта з новим значенням "hello world".

Методи рядків

Рядки мають різні вбудовані методи, які дозволяють маніпулювати і виконувати різін операції над ними. Нижче наведено деякі найпоширеніші методи роботи з рядками.

  • len(): повертає довжину рядка.
  • upper(): перетворює всі символи в рядку у верхній регістр.
  • lower(): перетворює всі символи в рядку на нижній регістр.
  • capitalize(): робить перший символ рядка великою.
  • title(): перетворює перший символ кожного слова на верхній регістр
  • strip(): видаляє пробіли на початку та в кін ці рядка.
  • lstrip(): видаляє початкові пробіли з рядка.
  • rstrip(): видаляє кінцеві пробіли з рядка.
  • find(): шукає підрядок і повертає індекс першого входження (або -1, якщо не знайдено).
  • split(): Розділяє рядок на список підрядків на основі розділювача.
  • count(): Підраховує входження підрядка в заданий рядок. Він повертає кількість разів, коли вказаний підрядок з'являється у вихідному рядку.
  • join(): використовується для об'єднання елементів послідовності (наприклад, списку) в один рядок, розділяючи їх заданим роздільником.

Більше інших методів для роботи з рядками можна знайти тут.

Особливості роботи з рядками

Робота з рядками є важливим аспектом програмування, оскільки рядки використовуються для представлення текстових даних. Python надає багатий набір вбудованих функцій і методів для ефективної роботи з рядками. Ось деякі з ключових функцій та операцій для роботи з рядками в Python:

  • Створення рядка: щоб створити рядок, просто візьміть текст у одинарні ('') або подвійні ("") лапки. Для багаторядкових рядків можна використовувати потрійні лапки ('''' ''').
  • Об'єднання рядків: Ви можете конкатенувати об'єднати два або більше рядків за допомогою оператора +.
  • Екранування рядка: Для включення спеціальних символів до рядка можна використовувати екрануючі послідовності (наприклад, \n для переходу на новий рядок, \t для табуляції).
  • Перевірка рядка: за допомогою ключового слова можна перевірити, чи містить рядок певний підрядок in.

Зріз рядка

Ви можете використовувати зріз для вилучення підрядка (або фрагмента рядка) з рядка. Зріз дозволяє витягти частину рядка, вказавши початковий і кінцевий індекси. Базовий синтаксис фрагментації рядка наступний:

start_index: Індекс першого символу, який ви хочете включити до зрізу. Він є інклюзивним, що означає, що символ з цим індексом буде включено до результату.

end_index: Індекс першого символу, який ви хочете виключити зі зрізу. Він є виключним, що означає, що символ з цим індексом не буде включено до результату.

Ось приклад використання зрізу для отримання рядка (підрядка) з рядка:

Не забувайте бути обережними при розбитті рядків на частини, щоб уникнути помилок виходу індексів за межі діапазону. Якщо start_index або end_index знаходиться поза допустимим діапазоном індексів рядка, Python згенерує помилку IndexError.

Форматований вивід

Форматування рядків можна виконати за допомогою різних методів і прийомів. Нижче наведено приклади використання:

  • F-рядки дозволяють вставляти вирази у фігурних дужках {} всередині рядка. Вирази всередині фігурних дужок буде обчислено і замінено їхніми значеннями.
  • Метод .format() можна використовувати для вставки змінних або виразів у рядок за допомогою заповнювачів {}.
  • Оператор % можна використовувати для форматування рядків, вказуючи заповнювачі %s для рядків і %d для цілих чисел (Старий спосіб).
  • Використання методу str.format() з позиційними аргументами, цей метод схожий на метод .format(), але він використовує числові індекси для визначення порядку аргументів.
  • Використання методу str.format() з іменованими аргументами цей підхід дозволяє використовувати іменовані заповнювачі у рядку форматування.
  • Використання конкатенації рядків (не рекомендується для складного форматування), хоча це не найефективніший і не рекомендований спосіб для складного форматування, ви можете використовувати конкатенацію рядків для створення відформатованих рядків.
